## 🔹 private フォルダ（非公開用）に入れるもの
ここは GitHub Pages には出さないので、自分のメモや機密を置けます。  

### 例: `/data/Obsidian/vault/private/secret.md`
# 秘密メモ

これは **private** フォルダに置いたファイルです。  
MkDocs の公開対象には含めないので、Webには出ません。
